Step-by-step start of emulation:

1. Download the source rom or bin file of the game/application that supports the Intel 8080 processor (the testing files are already in the Games folder)
2. Go to the Program folder and find the file Main.exe
3. Open the game/application file using Main.exe (for example, drag it to Main.exe )


Keys for interacting with the emulated software:

1. 'C' - is used to intercept control
2. '1' - player 1
3. '2' - player 2
4. 'W'
5. 'A'
6. 'D'
7. 'F'
8. 'T'
9. 'H'
